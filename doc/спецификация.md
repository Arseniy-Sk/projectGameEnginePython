# ТЕХНИЧЕСКАЯ СПЕЦИФИКАЦИЯ

## 1. ОБЩИЕ СВЕДЕНИЯ

### 1.1 Наименование продукта
**GameLib Framework** (далее – Фреймворк)

### 1.2 Назначение
Фреймворк предназначен для создания 2D игр на языке программирования Python с использованием библиотеки Tkinter.

### 1.3 Область применения
- Разработка казуальных 2D игр
- Образовательные проекты по геймдизайну
- Прототипирование игровых механик
- Инди-разработка

### 1.4 Платформа
- **Операционная система:** Windows 7/10/11
- **Интерпретатор Python:** версия 3.7 и выше
- **Графическая библиотека:** Tkinter (входит в стандартную поставку Python)

### 1.5 Зависимости
| Зависимость | Версия | Назначение |
|-------------|--------|-------------|
| Python | ≥3.7 | Среда выполнения |
| Pillow | ≥8.0.0 | Работа с изображениями |
| pydub | ≥0.25.0 | Обработка аудио |
| ffmpeg | (опционально) | Поддержка MP3 |

```

├── core/
```

## 3. КОМПОНЕНТЫ И ИНТЕРФЕЙСЫ

### 3.1 Ядро системы

#### 3.1.1 Класс `Engine`

**Назначение:** Управление игровым циклом и потоками.

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `running` | bool | Флаг работы движка |
| `game_loop_thread` | threading.Thread | Поток игрового цикла |
| `game` | Game | Ссылка на объект игры |
| `loop_delay` | float | Задержка между кадрами (сек) |

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| `__init__` | game: Game = None | - | Конструктор |
| `set_game` | game: Game | None | Установка игры |
| `start_game_loop` | - | None | Запуск игрового цикла |
| `stop_game_loop` | - | None | Остановка игрового цикла |
| `set_fps` | fps: int | None | Установка FPS |

#### 3.1.2 Класс `Game`

**Назначение:** Базовый класс для игр.

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `window` | main_game_window | Главное окно |
| `engine` | Engine | Ссылка на движок |

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| `__init__` | title: str, width: int, height: int, bg: str | - | Конструктор |
| `set_engine` | engine: Engine | None | Установка движка |
| `setup` | - | None | Инициализация (переопределяется) |
| `start` | - | None | Запуск игры |
| `update` | - | None | Обновление логики (переопределяется) |
| `end` | - | None | Завершение (переопределяется) |

### 3.8 Система событий

#### 3.8.1 Класс `Event`

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `source` | object | Источник события |
| `type` | str | Тип события |
| `data` | any | Данные события |

#### 3.8.2 Класс `EventBus`

**Назначение:** Шина событий для обмена сообщениями.

| Метод | Параметры | Описание |
|-------|-----------|----------|
| `subscribe` | source, event_type, callback | Подписка на события |
| `unsubscribe` | source, event_type, callback | Отписка |
| `emit` | source, event_type, data | Отправка события |

**Паттерны подписки:**
- `(source, event_type)` - точное совпадение
- `(source, None)` - все события от источника
- `(None, event_type)` - все события типа
- `(None, None)` - все события

**Глобальный экземпляр:** `global_bus`

## 4. ТРЕБОВАНИЯ К ИНТЕРФЕЙСАМ

### 4.1 Пользовательский интерфейс

Фреймворк не предоставляет стандартного пользовательского интерфейса. Интерфейс создается разработчиком с использованием GUI компонентов фреймворка.

### 4.2 API интерфейсы

Фреймворк экспортирует следующие публичные API:

```python
# Основные классы
from gamelib import Engine, Game, Scene, SceneSwitcher
from gamelib import GameObject, gameObject
from gamelib import InputManager, Camera2D, Vector2

# Компоненты
from gamelib import Component
from gamelib import BasePhisicComponent, Box_collider
from gamelib import AudioSource, AudioListener, ImageComponent

# GUI
from gamelib.core.model.gui.gui_factory import Label, Button, Panel
from gamelib.core.model.gui.gui_factory import CheckBox, Slider, DropDown, TextInput

# Системные
from gamelib import global_audio, global_bus
```
